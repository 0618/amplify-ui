One of the goals of amplify is to be a good citizen of the environment it exists in. As such amplify works hard to not interfere with other tools that are being used.

## CSS In JS

There are many CSS in JS libraries out there that give a number of benefits. Amplify does not use a CSS In JS library to implement its theming system but operates well with the major CSS In JS libraries out there.
