The default Amplify UI theme comes with a dark mode override by default.

To enable dark mode in your application set the `colorMode` prop in the `AmplifyProvider` to either:

- `system` if you want the user's OS color mode setting to be used
- `dark` if you want to explicitly set the app to dark mode
